# ⚖️ Calculateur-IMC

> 🚧 **Projet en cours de développement**

## 🛠️ Présentation
**Calculateur-IMC** est une application mobile moderne développée avec le framework **Flutter**. Son objectif est de fournir un outil simple et élégant pour calculer l'Indice de Masse Corporelle (IMC).

## 💻 Stack Technique
- **Framework** : **Flutter SDK**
- **Langage** : **Dart**

## ⚙️ Installation et Exécution

Pour installer le projet localement, assurez-vous d'avoir configuré l'environnement Flutter.

1.  **Cloner le dépôt** :
    ```sh
    git clone https://github.com/leul-mulugeta/calculateur-imc.git
    cd calculateur-imc
    ```

2.  **Installer les dépendances** :
    ```sh
    flutter pub get
    ```

3.  **Lancer l'application** :
    ```sh
    flutter run
    ```
